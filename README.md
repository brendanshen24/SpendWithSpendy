## Inspiration
In a digital world, kids are losing the tangible connection to money. We wanted to help parents teach financial responsibility through engaging visualizations and a physical object tied to spending.

## What it does
Spending with Spendy is a mobile app that offers engaging and intuitive money visualizations, a robust and secure reloading system, and a tangible physical object (NFC card) tied to spending money online.

## How we built it
We built an iOS React Native app with NFC integration. The app reads a special NFC card to create a physical connection to digital spending.
